# Сайт-биография Hunter Voronov (@swatergmail)

Современный многостраничный сайт с черным фоном и анимациями для специалиста в области киберразведки и OSINT.

## Особенности

- Адаптивный дизайн для всех устройств
- Плавные анимации и переходы
- Многоуровневая структура с навигацией
- Интерактивные элементы
- Оптимизирован для GitHub Pages

## Структура

- Главная страница с анимированным появлением ника
- Раздел "Обо мне" с информацией и навыками
- Раздел "Работы" с описанием громких кейсов
- Раздел "Проект Epysium" с описанием платформы
- Раздел "Кенты" с информацией о друзьях
- Футер с контактами и информацией о комьюнити

## Используемые технологии

- HTML5
- CSS3 (Flexbox, Grid, анимации)
- JavaScript (ES6+)
- Google Fonts (Inter, JetBrains Mono)
- Font Awesome иконки
