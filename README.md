# Сайт-биография Hunter Voronov (@swatergmail)

Современный многостраничный сайт с черным фоном и анимациями для специалиста в области киберразведки и OSINT.

## Внесённые изменения

1. **Убрана надпись "В комьюнити с 2022"** из верхней панели навигации (осталась только в футере)
2. **Исправлена стрелочка на мобильных устройствах** - теперь она не наезжает на текст
3. **Добавлена шкала 1/10 для навыков** с визуальным разделением на 10 частей
4. **Обновлены названия работ** в стиле комьюнити:
   - "Белый якорь", "Черный интернет", "Белый аист"
   - "Черный феникс", "Белая стена", "Черная тень"
   - Добавлены термины: "снос", "ликвидация", "деанонимизация"
5. **Статус проекта изменён на "Утерян"** с красным цветом и поясняющей заметкой

## Особенности

- Адаптивный дизайн для всех устройств
- Плавные анимации и переходы
- Многоуровневая структура с навигацией
- Интерактивные элементы
- Оптимизирован для GitHub Pages

## Структура

- Главная страница с анимированным появлением ника
- Раздел "Обо мне" с информацией и навыками (шкала 1/10)
- Раздел "Работы" с описанием громких кейсов (обновлённые названия)
- Раздел "Проект Epysium" со статусом "Утерян"
- Раздел "Кенты" с информацией о друзьях
- Футер с контактами и информацией о комьюнити

## Используемые технологии

- HTML5
- CSS3 (Flexbox, Grid, анимации)
- JavaScript (ES6+)
- Google Fonts (Inter, JetBrains Mono)
- Font Awesome иконки
